Módulo 1 

-Variaveis são um espaço de memoria para guardar uma certa informação e ela pode ser variavel ou constante;
-Algoritimos são uma sequencia de passos finitas para começar e terminar uma ação;
-Tipos de dados : 
		  - Nome de pessoas/caracter
		  - Idade ou numeros/inteiros
		  - Valor com pontos decimais/real
		  - Verdadeiro ou falso/lógico ou booleano
-Todo sistema é divido em entrada , processamento e saída
-Existem os operadores padrões sendo eles:
		  - Operador de Atribuição (=)
		  - Operador Aritméticos ( +, -,*,/,**) 
		  - Operadores Relacionais ( == , < , > , != , >= , <= )
		  - Operadores Lógicos (or , not).

Módulo 3

-Variaveis são tipos de dados que sao armazenados na memoria e podem ser reaproveitados;
-Em python uma variavel é sempre representada por um objeto sendo uma referencia;
-Como python é uma linguagem interpretada não é necessário declarar o tipo da variavel;
-Existem 4 principais tipos de dados:
		  - Inteiro(int) : armazena número inteiros;
		  - Real(float) : representa todos os números reais;
		  - Booleano(bool) : armazena valores booleanos (False/True);
		  - String(str) : armazena qualquer tipo de carcter entretanto que esteja em aspas;
-Para entrada de dados fornecidada pelo o usuario em python utilizamos o input;
-Nome = input("digite seu nome: ");
-Por padrao a função input considera os dados recebidos como string entao para converter esses dados se forem necessario utilizamos int(input("digite o numero"));
-Atalhos para auxiliar na formatação:
		  - \n pula a linha;
		  - \t insere tabulação horizontal;		
		  - \v insere tabulação vertical;
		  - \r equivale a tecla entre;
